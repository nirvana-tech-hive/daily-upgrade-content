# Contributing to Daily Upgrade Content Engine

Thank you for your interest in contributing to the **Daily Upgrade Content Engine**! This document outlines how you can contribute to the project.

## How to Contribute

### 1. Reporting Bugs
If you find a bug, please open an issue on GitHub with a clear description of the problem, including steps to reproduce it.

### 2. Suggesting Enhancements
If you have an idea for a new feature or improvement, open an issue on GitHub and describe your suggestion in detail.

### 3. Submitting Pull Requests
To contribute code or content:
1. Fork the repository.
2. Create a new branch for your changes.
3. Make your changes and test them thoroughly.
4. Commit your changes with a clear and descriptive message.
5. Push your changes to your fork.
6. Open a pull request to the `main` branch of this repository.

### 4. Writing Content
If you want to contribute blog posts, infographics, or support content:
- Follow the templates provided in the `content/templates` directory.
- Ensure your content aligns with the **Counter-AI Movement** ideology.
- Add your content to the appropriate directory and update the content calendar if necessary.

### 5. Code of Conduct
Please follow our [Code of Conduct](CODE_OF_CONDUCT.md) in all interactions with the project.

## Development Setup

### Prerequisites
- Python 3.10 or higher
- Git

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/nirvana-tech-hive/daily-upgrade-content.git
   cd daily-upgrade-content
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

### Running Scripts
- To generate HTML posts:
  ```bash
  python automation/scripts/generate_html.py
  ```
